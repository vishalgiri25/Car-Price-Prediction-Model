# Car-Price-Prediction-Model
This project focuses on predicting the price of cars based on various features such as seller type, kms driven, furl type, and other specifications. The model has been built using Linear Regression and Lasso Regression, leveraging their ability to capture relationships between numerical features and the target variable.

The dataset was preprocessed with feature scaling, encoding of categorical variables, and handling of missing values to ensure optimal performance.
The Linear Regression model serves as the baseline, while Lasso Regression helps in feature selection and reducing overfitting.

This repository includes:

    Cleaned dataset 

    Python code for data preprocessing and model training

    Model evaluation metrics for accuracy comparison

This project demonstrates a simple yet effective approach to machine learning
